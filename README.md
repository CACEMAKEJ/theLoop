# The Loop: An Exploration of Doomscrolling and FOMO

The Feedback Loop is a minimalist, text-focused website designed to raise awareness about the hidden effects of user-tailored content and social media design.
It explains how algorithms shape attention, why doomscrolling occurs, and how fear of missing out affects mental well-being.
